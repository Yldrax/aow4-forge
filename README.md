# Pantheon Forge
### AoW4 Item Forge Automation for Convenient Pantheon Point Farming

## Features
- Craft all your essence into tier I daggers
- Optionally disenchant your whole arsenal
- Automatic switching between forging and disenchanting until ressources are empty

## Installation Instructions
1. Download the latest release .zip file
2. Unpack .zip file into folder of your choice
3. Open Pantehon Forge exe

## Usage Guide
1. Open Program
2. Optionally activate disenchanting
3. Optionally change the speed (On my PC opening the crafting options takes a little bit of time, might be faster or slower on your pc, default 3 works for me
4. Navigate to the Item Forge Screen in your AoW4 Game
5. Press Start

Press the Escape Key (esc) at any time to cancel

Emergency PyautoGUi Failsafe is enabled - move mouse to any screen corner to stop
